Instrucciones para configurar Docker, MariaDB, cliente MySQL y DBeaver en Alpine Linux y máquinas Hyper-V

Paso 1: Instalar Docker en Alpine Linux

  1. Actualiza los repositorios e instala Docker:
    2. sudo apk update
    sudo apk add docker
  2. Arrancar Docker
    3. sudo service docker start
  3. Verifica que Docker está instalado correctamente:
    4. docker --version

Paso 3: Instalar DBeaver en tu máquina local

  1. Dirígete a la página oficial de DBeaver: https://dbeaver.io/download/
  2. Descarga e instala la versión correspondiente a tu sistema operativo.

Paso 4: Instalar MySQL Client en Alpine Linux (si deseas acceder desde la terminal)

  1. Si deseas conectarte a MariaDB desde la terminal usando el cliente MySQL, instala el cliente de MySQL con el siguiente comando:
    2. sudo apk add mysql-client
  2. Conéctate al contenedor de MariaDB con el cliente MySQL (reemplaza <IP_DEL_CONTENEDOR> por la IP del contenedor o máquina):
    3. mysql -h <IP_DEL_CONTENEDOR> -u root -p
  3. Luego ingresa la contraseña de root cuando se te solicite.

Paso 5: Conectarse a MariaDB desde DBeaver

  1. Abre DBeaver e ingresa los datos de conexión:
    • Host: Dirección IP de la máquina virtual donde corre MariaDB (debe ser la IP de la máquina de Hyper-V).
    • Puerto: 3306
    • Usuario: root
    • Contraseña: root (la contraseña que configuraste).
  2. Haz clic en "Test Connection" para verificar la conexión y luego en "Finish" para completar.

Paso 6: Conectarse al MariaDB de tus compañeros

  1. En la máquina virtual de Hyper-V, abre la configuración de la máquina virtual.
  2. Selecciona "Adaptador de red" y elige "Conmutador externo" en la configuración de la máquina virtual.
  3. Configura una IP estática para la máquina virtual (asegúrate de que esté en la misma red que la máquina local).
  4. Asegúrate de que tu máquina local puede acceder a esa IP.

Paso 7: Obtener la IP de la máquina del compañero

Para saber la IP de la máquina virtual de tu compañero, puedes usar el siguiente comando dentro de la máquina virtual de Hyper-V:

ip a

Este comando te mostrará la dirección IP de la máquina virtual. Debes buscar la interfaz de red que esté conectada a la red externa, normalmente tendrá una IP como 192.168.x.x o similar.

¡Listo!

Ahora deberías estar conectado a la base de datos MariaDB de tu compañero desde DBeaver, o desde la terminal con el cliente MySQL.
